## 1. Java 언어 특징

Java는 객체지향 프로그래밍 언어이다.

객체지향 언어는 프로그램을 객체 단위로 구성하고, 객체가 가진 데이터와 기능을 중심으로 프로그램을 설계하는 방식이다.

Java는 소스 코드를 바로 실행하지 않고, 먼저 컴파일 과정을 거친다.

기본 흐름은 다음과 같다.

```text
Java 소스 코드
→ 컴파일
→ 바이트코드
→ JVM에서 실행
```

Java는 컴파일 언어의 특징과 인터프리터 언어의 특징을 함께 가진다.

컴파일러는 Java 소스 코드를 JVM이 이해할 수 있는 바이트코드로 변환한다.

그 후 JVM은 바이트코드를 실행한다.

이 구조 덕분에 Java는 특정 운영체제에 직접 종속되지 않고, JVM이 설치된 환경이라면 실행될 수 있다.

정리하면 다음과 같다.

- Java는 객체지향 언어이다.
- Java 소스 코드는 바이트코드로 컴파일된다.
- 바이트코드는 JVM 위에서 실행된다.
- JVM이 운영체제 차이를 중간에서 처리해준다.

## 2. 주석

주석은 프로그램 실행에 영향을 주지 않는 설명 문장이다.

코드를 작성한 이유나 동작 방식을 설명하기 위해 사용한다.

혼자 개발할 때도 도움이 되지만, 협업할 때 다른 사람이 코드를 이해하는 데 특히 중요하다.

Java의 대표적인 주석은 다음과 같다.

- 한 줄 주석: `//`
- 여러 줄 주석: `/* */`

한 줄 주석은 `//`를 사용한다.

```java
// 한 줄 주석
```

여러 줄 주석은 `/* */`를 사용한다.

```java
/*
 * 여러 줄 주석
 */
```

주석은 코드의 의도를 설명하기 위해 사용해야 한다.

단순히 코드 내용을 그대로 반복하는 주석보다는, 왜 이렇게 작성했는지 설명하는 주석이 더 좋다.

## 3. 변수

변수는 데이터를 저장하는 공간에 이름을 붙인 것이다.

프로그램에서 자주 사용하는 값, 사용자 입력값, 연산 결과 등을 저장할 때 사용한다.

Java에서 변수를 사용하려면 먼저 자료형과 변수명을 선언해야 한다.

```java
int age = 20;
double height = 175.5;
boolean isStudent = true;
```

여기서 `=`는 대입 연산자이다.

오른쪽 값을 왼쪽 변수에 저장한다.

정리하면 다음과 같다.

- 변수는 값을 저장하는 공간이다.
- Java 변수는 자료형을 지정해야 한다.
- `=`는 값을 변수에 대입하는 연산자이다.

## 4. 기본 자료형

Java의 기본 자료형은 크게 다음과 같이 나눌 수 있다.

- 정수형
- 실수형
- 논리형
- 문자형

### 정수형

정수형은 소수점이 없는 숫자를 저장한다.

대표적인 정수형 타입은 다음과 같다.

- `byte`
- `short`
- `int`
- `long`

`byte`는 8bit 크기의 정수형 타입이다.

8bit는 2의 8승만큼의 정보를 표현할 수 있다.

일반적으로 정수는 `int`를 가장 많이 사용한다.

### 실수형

실수형은 소수점이 있는 숫자를 저장한다.

대표적인 실수형 타입은 다음과 같다.

- `float`
- `double`

일반적으로 실수는 `double`을 많이 사용한다.

### 논리형

논리형은 참과 거짓을 저장한다.

Java에서는 `boolean`을 사용한다.

값은 `true` 또는 `false`이다.

### 문자형

문자형은 문자 하나를 저장한다.

Java에서는 `char`를 사용한다.

문자 값은 작은따옴표로 감싼다.

```java
char grade = 'A';
```

## 5. 형변환

형변환은 값의 자료형을 다른 자료형으로 바꾸는 것이다.

형변환은 크게 자동 형변환과 명시적 형변환으로 나눌 수 있다.

### 자동 형변환

자동 형변환은 작은 타입의 값을 큰 타입에 넣을 때 발생한다.

작은 값을 더 큰 공간에 넣는 것이므로 Java가 자동으로 처리할 수 있다.

```java
int number = 10;
double result = number;
```

위 코드에서 `int` 값은 `double` 타입으로 자동 변환된다.

쉽게 말하면 작은 그릇의 값을 큰 그릇에 넣는 경우이다.

### 명시적 형변환

명시적 형변환은 개발자가 직접 타입 변환을 지정하는 것이다.

큰 타입의 값을 작은 타입에 넣을 때는 데이터 손실이 발생할 수 있기 때문에 자동으로 변환되지 않는다.

이때는 개발자가 직접 형변환을 작성해야 한다.

```java
double height = 175.8;
int value = (int) height;
```

위 코드에서 `175.8`은 `175`가 된다.

소수점 이하 값이 사라지므로 데이터 손실이 발생한다.

정리하면 다음과 같다.

- 자동 형변환: 작은 타입에서 큰 타입으로 변환
- 명시적 형변환: 큰 타입에서 작은 타입으로 강제 변환
- 명시적 형변환은 데이터 손실에 주의해야 한다.

## 6. 연산자

연산자는 값을 계산하거나 비교하거나 대입할 때 사용하는 기호이다.

연산자에는 우선순위가 있다.

여러 연산자가 함께 사용되면 우선순위에 따라 먼저 계산되는 연산이 달라진다.

대표적인 연산자 종류는 다음과 같다.

- 단항 연산자
- 이항 연산자
- 산술 연산자
- 대입 연산자
- 비교 연산자
- 논리 연산자
- 증감 연산자
- 논리 반전 연산자
- 형변환 연산자

### 단항 연산자

단항 연산자는 하나의 값에 대해 연산한다.

예를 들어 부호 연산자, 증감 연산자, 논리 반전 연산자가 있다.

```java
int number = 10;
number++;
```

### 이항 연산자

이항 연산자는 두 개의 값에 대해 연산한다.

예를 들어 산술 연산자, 비교 연산자, 논리 연산자가 있다.

```java
int result = 10 + 20;
```

### 산술 연산자

산술 연산자는 숫자 계산에 사용한다.

- `+`
- `-`
- `*`
- `/`
- `%`

### 대입 연산자

대입 연산자는 오른쪽 값을 왼쪽 변수에 저장한다.

```java
int age = 20;
```

### 비교 연산자

비교 연산자는 두 값을 비교하고 결과를 `true` 또는 `false`로 반환한다.

- `==`
- `!=`
- `>`
- `<`
- `>=`
- `<=`

### 논리 연산자

논리 연산자는 여러 조건을 함께 판단할 때 사용한다.

- `&&`: AND
- `||`: OR
- `!`: NOT

연산자 우선순위가 헷갈릴 때는 괄호를 사용해 의도를 명확히 표현하는 것이 좋다.

## 7. 조건문

조건문은 조건에 따라 실행할 코드를 선택할 때 사용한다.

프로그램은 기본적으로 위에서 아래로 순차적으로 실행된다.

하지만 조건문을 사용하면 특정 조건에 따라 다른 흐름으로 실행할 수 있다.

대표적인 조건문은 다음과 같다.

- `if`
- `else if`
- `else`
- `switch`

### if문

`if`문은 조건식이 `true`일 때만 특정 코드를 실행한다.

```java
if (score >= 60) {
    System.out.println("합격");
}
```

### else if문

`else if`는 여러 조건을 순서대로 검사할 때 사용한다.

```java
if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

### switch문

`switch`문은 하나의 값에 따라 여러 분기 중 하나를 선택할 때 사용한다.

```java
switch (gender) {
    case "M":
        System.out.println("남성");
        break;
    case "F":
        System.out.println("여성");
        break;
    default:
        System.out.println("알 수 없음");
        break;
}
```

Java에서 문자열을 `switch`에 사용할 때는 문자열 값 비교로 동작한다고 이해하면 된다.

다만 `switch` 대상 값이 `null`이면 `NullPointerException`이 발생할 수 있으므로 주의해야 한다.

## 8. 반복문

반복문은 같은 작업을 여러 번 실행할 때 사용한다.

대표적인 반복문은 다음과 같다.

- `while`
- `for`

### while문

`while`문은 조건식이 `true`인 동안 반복 실행된다.

조건식이 처음부터 `false`이면 한 번도 실행되지 않을 수 있다.

```java
int i = 0;

while (i < 5) {
    System.out.println(i);
    i++;
}
```

### for문

`for`문은 초기화, 조건식, 증감식, 실행 코드로 구성된다.

반복 횟수가 정해져 있을 때 자주 사용한다.

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

### 반복문 중첩

반복문 안에 또 다른 반복문을 작성할 수 있다.

이를 반복문 중첩이라고 한다.

예를 들어 별 찍기, 2차원 배열 순회 등에서 자주 사용된다.

## 9. 탈출문

탈출문은 반복문이나 메서드의 실행 흐름을 중간에 제어할 때 사용한다.

대표적인 탈출문은 다음과 같다.

- `break`
- `continue`
- `return`

### break

`break`는 반복문이나 `switch`문을 빠져나갈 때 사용한다.

```java
while (true) {
    break;
}
```

중첩 반복문에서는 `label`을 사용해 특정 반복문까지 한 번에 빠져나갈 수 있다.

### continue

`continue`는 현재 반복을 건너뛰고 다음 반복으로 넘어갈 때 사용한다.

```java
for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue;
    }

    System.out.println(i);
}
```

위 코드에서 `i`가 2일 때는 출력하지 않고 다음 반복으로 넘어간다.

### return

`return`은 메서드 실행을 종료하고 값을 반환할 때 사용한다.

반환값이 없는 메서드에서는 `return;`만 사용할 수 있다.

반환값이 있는 메서드에서는 메서드의 반환 타입에 맞는 값을 반환해야 한다.

## 10. 오늘 정리

오늘은 Java 기초 문법을 다시 복습했다.

가장 중요한 흐름은 다음과 같다.

```text
Java 언어 특징 이해
→ 변수와 자료형 이해
→ 형변환 이해
→ 연산자 이해
→ 조건문으로 흐름 제어
→ 반복문으로 반복 처리
→ break, continue, return으로 흐름 탈출
```

특히 형변환은 자동으로 되는 경우와 명시적으로 작성해야 하는 경우를 구분하는 것이 중요하다.

조건문과 반복문은 프로그램의 흐름을 제어하는 기본 문법이므로, 예제를 직접 작성하면서 익숙해질 필요가 있다.

오늘 학습에서 가장 중요한 점은 Java 문법을 단순히 암기하는 것이 아니라, 값이 어떤 타입으로 저장되고 어떤 흐름으로 실행되는지 이해하는 것이다.
