# codeit-bootcamp-til

코드잇 부트캠프 기간 동안의 학습 기록 저장소입니다.  
TIL, WIL, 예습·복습, 과제, 알고리즘, 프로젝트 회고를 정리합니다.

## 📌 작성 목적

이 저장소는 단순히 공부한 내용을 기록하는 것이 아니라,  
부트캠프 과정에서 내가 무엇을 배웠고, 어떤 부분에서 어려움을 겪었으며,  
어떻게 해결하고 성장했는지를 남기기 위해 작성합니다.

## 🗓️ 학습 기간

- 시작일: 2026년 4월 30일
- 과정: AWS 활용 Spring 백엔드 개발자 실무 부트캠프 13회차

## 📁 폴더 구조

```text
codeit-bootcamp-til/
├── README.md
├── daily/        # 매일 작성하는 TIL
├── weekly/       # 매주 작성하는 WIL
├── learning/     # 부트캠프 강의, 예습 및 복습 정리
├── assignments/  # 부트캠프 과제 정리
├── algorithm/    # 알고리즘 문제 풀이 기록
└── projects/     # 프로젝트 회고 및 진행 기록
```

## ✍️ 작성 방식

### TIL

매일 학습한 내용을 정리합니다.

- 오늘 한 일
- 오늘 배운 것
- 어려웠던 점
- 해결 방법
- 내일 할 일

### WIL

한 주 동안의 학습을 돌아보고 정리합니다.

- 이번 주에 한 일
- 이번 주에 배운 것
- 어려웠던 점
- 개선할 점
- 다음 주 계획

## 📚 학습 기록 기준

학습 기록은 단순한 강의 내용 복사가 아니라,  
내가 이해한 내용과 직접 겪은 시행착오를 중심으로 작성합니다.

가능하면 아래 내용을 포함합니다.

- 내가 이해한 개념
- 헷갈렸던 부분
- 직접 작성한 코드
- 오류가 발생한 이유
- 해결 과정
- 다음에 다시 볼 내용

## ✅ 목표

- 매일 꾸준히 학습 기록 남기기
- 배운 내용을 내 언어로 정리하기
- 모르는 부분을 명확히 파악하기
- 프로젝트와 과제 경험을 기록으로 남기기
- 취업 준비 시 성장 과정을 보여줄 수 있는 자료 만들기
